# 主要画面一覧｜人生の羅針盤
PRD「きもち名言」をベースに、必要な画面とその機能を洗い出したもの。
このアプリは **1画面で完結する設計**（MVP）。そのため「画面」とは別ページのことではなく、
**1枚のページが切り替わる4つの"状態"** を指す。HTML/CSS/JS（jQuery）で、各状態を出し分けして実現する。
> 

---

## 画面遷移の全体像

```
┌─────────────┐
│ ①入力画面   │ ← 最初に表示される
└─────┬───────┘
      │ 送信ボタン or Enter
      ▼
┌─────────────┐
│ ②ローディング│ ← AI・APIの往復を待つ（数秒）
└─────┬───────┘
      │
   ┌──┴──┐
 成功     失敗
   │        │
   ▼        ▼
┌──────┐ ┌──────┐
│③結果 │ │④エラー│
└──┬───┘ └──┬───┘
   │         │
   └────┬────┘
   「やり直し / もう一度」
        │
        ▼
   ①入力画面 に戻る
```

### ② ローディング画面（探しています…）

| 項目 | 内容 |
| --- | --- |
| 目的 | AI①→名言API→AI②の往復に数秒かかるため、待っていることを伝える（白画面で固まらせない） |
| 表示する要素 | 「言葉を探しています…」などのメッセージ ／ 動きのある待機表示（スピナー等） |
| 機能・挙動 | ・処理中は入力欄と送信ボタンを操作できないようにする（二重送信の防止） |
| 次の状態へ | 全部成功 → ③ 結果 ／ どこかで失敗 → ④ エラー |

| --- | --- |

### ④ エラー画面

| 項目 | 内容 |
| --- | --- |
| 目的 | 失敗しても固まらず、もう一度試せるようにする |
| 表示する要素 | やさしめのエラーメッセージ ／「もう一度」ボタン |
| 想定するエラーの種類 | ・名言API（Quotable）が不通<br>・AIがJSON以外を返してきて読み取れない<br>・通信そのものの失敗（ネットワーク等） |
| 機能・挙動 | ・どの失敗でも、白画面ではなくこの状態を表示する |
| 次の状態へ | もう一度 → ① 入力画面（または直前の気持ちで再実行） |

## 機能一覧（画面をまたぐ処理）

画面の見た目とは別に、裏側で動く機能を一覧にしたもの。後で「どの機能をどの関数にするか」を決める土台になる。

| # | 機能 | 主に動く画面 | ざっくりの中身 |
| --- | --- | --- | --- |
| 1 | 気持ちの入力受け取り | ① | テキスト入力欄の値を取り出す |
| 2 | 入力バリデーション | ① | 空かどうかをチェックする |
| 3 | 送信のトリガー | ①→② | ボタンクリック等で処理を開始する |
| 4 | AI①呼び出し（タグ選択） | ② | 気持ちを渡し、実在タグから1つ選ばせる |
| 5 | JSONの安全な読み取り | ② | AI①の返事から `{ }` を拾って tag を取り出す |
| 6 | 名言API呼び出し | ② | 選ばれたタグで名言を1つ取得する |
| 7 | AI②呼び出し（翻訳＋やさしい一言） | ② | 英語名言を和訳し、寄り添う一言を添える |
| 8 | 画面状態の切り替え | 全体 | 入力／ローディング／結果／エラーを出し分ける |
| 9 | 結果の表示 | ③ | 訳・原文・作者・一言を画面に流し込む |
| 10 | エラーハンドリング表示 | ④ | 失敗を受け止めてエラー画面を出す |
| 11 | やり直し（リセット） | ③④→① | 表示を消し、入力画面に戻す |

## 実装の見取り図（任意・あとで決める用）

jQueryで「状態の出し分け」をするので、画面ごとに入れ物（id）を用意し、`hidden` クラスの付け外しで見せ隠しする想定。
※これは提案。実際のidや構造は自分で組みながら調整してOK。

| id（名札）の案 | 役割 | 属する画面 |
| --- | --- | --- |
| `#feeling-input` | 気持ちを書く入力欄 | ① |
| `#submit-btn` | 送信ボタン | ① |
| `#loading` | ローディングの入れ物（初期は隠す） | ② |
| `#result` | 結果の入れ物（初期は隠す） | ③ |
| `#quote-ja` | 名言の日本語訳 | ③ |
| `#quote-en` | 名言の原文（英語） | ③ |
| `#quote-author` | 作者名 | ③ |
| `#ai-note` | AIのやさしい一言 | ③ |
| `#retry-btn` | やり直しボタン | ③ |
| `#error` | エラーの入れ物（初期は隠す） | ④ |
| `#error-message` | エラー文言 | ④ |
| `#error-retry-btn` | もう一度ボタン | ④ |
